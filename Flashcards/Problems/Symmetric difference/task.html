Symmetric difference Open on JetBrains Academy

Implement a method for finding the symmetric difference of the two given sets. Elements in the resulting set can be in any order.

An example

The symmetric difference of two sets {1, 2, 3} and {0, 1, 2} is {0, 3}


Sample Input:
1 2 3
0 1 2

Sample Output:
0 3


Memory limit: 256 MB
Time limit: 8 seconds
